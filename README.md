# cn-portal-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式保存，不绑定任何特定域名或网站，可作为通用页面资源进行访问或引用。

## 项目简介

本项目旨在提供一组独立的 HTML 页面存档，适用于展示、测试、分享或快速部署。所有页面均为纯前端实现，不依赖后端服务，可直接在浏览器中打开。

## 目录说明

- `pages/`：存放所有 HTML 页面文件，每个文件对应一个独立页面。
- `assets/`：存放页面引用的静态资源（如 CSS、JS、图片等），按需组织。
- `index.html`：可选入口页面，用于导航或索引。

## 页面归档说明

- 页面内容以 HTML 格式保存，保持原始结构与样式。
- 归档过程中不修改页面内部链接或资源引用，确保本地或静态托管环境下可用。
- 每个页面独立存在，互不依赖，方便单独使用或复制。

## 使用方式

1. 直接打开 `pages/` 目录下的任意 HTML 文件即可查看页面内容。
2. 可将整个仓库部署到任意静态托管服务（如 GitHub Pages、Netlify 等）以在线访问。
3. 如需引用页面中的资源，请确保路径正确或使用相对路径。

## 维护说明

- 页面归档以“保持原样”为原则，不主动修改页面内部逻辑或内容。
- 欢迎提交 Pull Request 添加新的独立 HTML 页面，但请确保页面不含恶意代码或违反法律法规的内容。
- 如有页面失效、资源缺失等问题，可提交 Issue 说明。

## 贡献

欢迎通过 Issue 或 Pull Request 参与本仓库的维护。提交新页面时，请放置在 `pages/` 目录下，并在提交信息中简要说明页面用途。

## 许可证

本仓库内容按原样提供，不附带任何明示或暗示的保证。请自行评估页面内容的安全性及适用性。
